# Calculadora de IMC

Projeto web desenvolvido com HTML, CSS e JavaScript que calcula o Índice de Massa Corporal (IMC) a partir dos dados inseridos pelo usuário.

## Funcionalidades

- Inserção de peso e altura
- Validação de dados do usuário
- Cálculo automático do IMC
- Exibição do nível de classificação do IMC

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
